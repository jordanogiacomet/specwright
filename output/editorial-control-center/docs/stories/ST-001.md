# ST-001 — Define CMS content model (part 1 of 2)

**Story key:** `product.cms-content-model`

## Description

Define CMS content model — collections: pages, posts, authors, media; globals: site-settings, homepage.

## Acceptance Criteria

- [ ] Payload collection `pages` exists with fields `title`, `slug`, `hero`, `body_or_blocks`, `seo_title`, `seo_description` — Manage landing pages and evergreen public site content.
- [ ] `pages` includes a unique `slug` used by the later public route `/pages/[slug]` and preview lookup.
- [ ] `pages` can reference `media` for hero or inline assets.
- [ ] `pages` is modeled so the later draft/publish story can add editorial status handling without redefining the schema.
- [ ] Payload collection `posts` exists with fields `title`, `slug`, `excerpt`, `body`, `author`, `featured_image`, `published_at` — Manage news and editorial articles.
- [ ] `posts` includes a unique `slug` used by the later public route `/posts/[slug]` and preview lookup.
- [ ] `posts` includes `author` relation to `authors`, optional `featured_image` relation to `media`.
- [ ] `posts` is modeled so the later draft/publish story can add editorial status handling without redefining the schema.
- [ ] Payload collection `authors` exists with fields `name`, `slug`, `bio`, `avatar` — Manage author profiles referenced by posts.

## Scope Boundaries

- Do NOT implement draft/publish transitions, preview mode, or scheduled publishing in this story.
- Do NOT build media library UI or public page routes here — only define the schema and registration points.
- Do NOT add collections/globals beyond the ones named in this story unless `spec.json` is updated first.

## Expected Files

- `src/payload.config.ts (updated with collections and globals)`
- `src/collections/Pages.ts`
- `src/collections/Posts.ts`
- `src/collections/Authors.ts`

## Dependencies

- `bootstrap.backend`

## Validation

- Run: `npm run build`
- Manual: Admin can create sample page/post entries, assign author/media relations, and edit globals without missing fields.
