# ST-004 — Setup database

**Story key:** `bootstrap.database`

## Description

Configure postgres database with connection pool and migrations.

## Acceptance Criteria

- [ ] postgres connection is configured via environment variable
- [ ] Database client or ORM is installed and configured
- [ ] Initial migration or schema setup runs without errors
- [ ] Connection can be verified programmatically
- [ ] docker-compose.yml includes database service

## Scope Boundaries

- Do NOT create application tables yet — only the connection layer
- Do NOT add seed data
- Do NOT implement any API endpoints

## Expected Files

- `docker-compose.yml`
- `src/lib/db.ts`
- `.env.example (updated with DB connection string)`

## Dependencies

- `bootstrap.repository`

## Validation

- Run: `npm run build`
- Manual: Database container starts and connection succeeds
