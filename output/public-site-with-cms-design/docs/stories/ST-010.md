# ST-010 — Create frontend application

**Story key:** `bootstrap.frontend`

## Description

Implement frontend application shell using nextjs.

## Acceptance Criteria

- [ ] Frontend application renders using nextjs
- [ ] Root page loads without errors
- [ ] Basic layout component exists (shell, navigation placeholder)
- [ ] Next.js app router is configured with route groups

## Scope Boundaries

- Do NOT implement pages beyond the root layout and a placeholder home page
- Do NOT add authentication UI — that is a separate story
- Do NOT style extensively — basic layout only

## Expected Files

- `src/app/(app)/page.tsx`
- `src/components/Layout.tsx`
- `src/app/layout.tsx`

## Dependencies

- `bootstrap.backend`

## Validation

- Run: `npm run build`
- Manual: Frontend loads in browser without errors
